# gcu-avf

Documentation for the `gcu-avf` project. This is the main control system entry point for the GCU AV Framework. It wires together all framework layers — configuration, hardware, application logic, and user interface — into a single Crestron control system program.

---

## Core

| Type | Description |
|------|-------------|
| [ControlSystem](ControlSystem.md) | Main control system class; orchestrates the full startup and shutdown sequence for the GCU AV Framework. |
